## Membership
- in
	- 어떤 것이 어떤 것 안에 있는지
- not in
	- 어떤 것이 어떤 것 안에 없는지
- 예시
	```python
	>>> 'a' not in ['a', 'b', 'c']
	False
	>>> 'er' in 'berry'
	True
	```
