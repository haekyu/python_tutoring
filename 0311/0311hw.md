## 1. 함수 연습 1
- Factorial 함수를 구현해 보세요. 단! 재귀함수가 **아닌** 버전으로 구해보세요.
    - **input**: n
    - **output**: n!의 값

- n!은 다음과 같이 정의합니다.
    - n이 0보다 작거나 같은 경우: n! = 1
    - n이 0보다 큰 경우: n! = 1부터 n 까지의 곱

- 예) 
    ```python
    print(fac(0))
    print(fac(3))
    print(fac(5))
    ```
    출력 결과
    ```
    1
    6
    120
    ```
